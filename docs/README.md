1. 做当前版本决策时，优先以 [vNext 产品方案](./plan.md) 为准。
2. 需要复用旧视觉、旧交互或追溯历史判断时，再查 `legacy/`。
3. 若后续产生新的实施清单、数据模型设计或页面规范，应归入“当前有效”而不是继续追加到旧版文档。
